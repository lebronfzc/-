# 小红书封面创作：简单用法

## 你需要准备什么

- 一张参考封面图。
- 你想替换到封面上的新标题或文案。
- 如果参考封面里有真人脸，并且你想换成自己，还需要提供一张清晰的本人照片。

## 最简单的说法

直接这样发：

```text
用小红书封面创作，参考这张图，把标题改成：3个普通人也能用的AI写作方法
```

默认流程是：先按参考图做封面的画面和版式，再把你的中文标题本地排版加上。这样比直接让 AI 在图里生成中文更稳，不容易缺字、错字。

如果是人物封面，可以这样发：

```text
用小红书封面创作，参考这张封面的风格，用我的照片做主视觉。
标题：普通人如何做个人IP
副标题：从第一篇笔记开始
```

## 小提醒

- 参考图越清楚，做出来越接近你想要的风格。
- 标题尽量短一点，封面更容易读。
- 如果标题很长，可以直接给主标题、副标题，我会按封面层级拆开排版。
- 如果不想露脸，直接说明不要真人脸。
